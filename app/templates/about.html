What this tool does

This interface is a front-end for a local AI model fine-tuned on UK contract and consumer law. It aims to:

  • Use the IRAC method Issue · Rule · Application · Conclusion
  • Distinguish claim types Goods · Services · Digital content
  • Read your PDFs e.g. Consumer Rights Act 2015
  • Run locally Ollama – no cloud calls
Legal disclaimer

This is an AI research tool, not a solicitor and not a replacement for legal advice. Always:

  • Verify citations Check primary legislation and cases
  • Consult a professional For real disputes or important decisions
  • Check jurisdiction Model focuses on UK law only
Setup checklist
  • Clone/download repository Project on your machine
  • Install Python dependencies pip install -r requirements.txt
  • Download model file Place UK_Law_Real_Final.gguf in project folder
  • Create Ollama model ollama create uk-lawyer -f Modelfile
  • Pull embedding model ollama pull nomic-embed-text
  • Create legal_docs/ folder Add PDFs like CRA 2015, Tesco v USDAW 2024
  • Run the GUI streamlit run lawyer_gui.py
