Quick start
  • Install Python deps pip install -r requirements.txt
  • Download model file UK_Law_Real_Final.gguf
  • Create Ollama model ollama create uk-lawyer -f Modelfile
  • Embedding model ollama pull nomic-embed-text
  • Launch GUI streamlit run lawyer_gui.py
